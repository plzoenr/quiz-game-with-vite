# Install Instuction

## 1.Clone project

$ ```git clone https://github.com/plzoenr/quiz-game-with-vite.git```

## 2.Move to clone directory

$ ```cd quiz-game-with-vite```

## 3.Install additional package

$ ```npm install```

## 4.Start Sever by

$ ```npm run dev```

## 5.Copy Link and open in web browser

url: ```http://localhost:5173```
