# Install Instuction

## 1.Clone project

$`git clone https://github.com/plzoenr/quiz-game-with-vite.git`

## 2.Install additional package

$`npm install`

## 3.Start Sever by

$`npm run dev`

## 4.Copy Link and open in web browser

url: `localhost:5173`
